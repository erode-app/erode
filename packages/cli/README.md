# @erode-app/cli

Interactive CLI for detecting architecture drift. Analyzes pull requests and
local git diffs against your declared architecture model.

## Usage

```sh
npx @erode-app/cli check
```

Or install globally:

```sh
npm install -g @erode-app/cli
```
